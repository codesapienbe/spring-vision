Deep Java Library (DJL)

The Native AI Bridge for the Java Ecosystem

What is DJL?

The Deep Java Library (DJL) is a high-level, open-source library built by Amazon. Its core function is to simplify the development and deployment of deep learning models for Java developers. By acting as a unified abstraction layer over deep learning engines, DJL allows engineers to integrate cutting-edge AI (like PyTorch and TensorFlow) directly into their familiar Java applications, bypassing the complexity of environment and engine management.

900+ Ready Models

Models are available in the Model Zoo for immediate use and deployment.

Model Zoo Domain Distribution

The majority of models in the DJL Model Zoo are focused on Computer Vision (65%), providing robust tools for image and video analysis. NLP models form the second largest category, emphasizing the library's utility across multiple core AI domains.

DJL's Engine-Agnostic Architecture

This core design principle ensures a developer only interacts with one high-level API, decoupling their application logic from the underlying execution framework, guaranteeing flexibility and future-proofing.

Engine Abstraction

VisionTemplate: Deep Dive into Multi-Class Prediction (MCP)

The `VisionTemplate` bean simplifies complex MCP tasks by mapping them to high-level, production-grade DJL models. The models listed below are currently used or represent the target model type for the corresponding method.

🧑‍🔬

Face Detection

Locate and box faces in an image for preprocessing or authentication.

Method: detectFaces(bytes)

Core Model: RetinaFace

DJL Domain: Computer Vision (Detection)

📦

Object Detection

Identify, locate, and classify multiple instances of objects (e.g., cars, pets, furniture).

Method: detectObjects(bytes)

Core Model: YOLOv5

DJL Domain: Computer Vision (Detection)

🖼️

Image Classification

Assign a single, dominant label to an entire image (e.g., "landscape," "animal").

Method: classifyImage(bytes)

Core Model: ResNet-50

DJL Domain: Computer Vision (Classification)

🔤

Text Recognition (OCR)

Extract readable text from unstructured image data, such as documents or signs.

Method: recognizeText(bytes)

Core Model: PaddleOCR

DJL Domain: Computer Vision / NLP

🤸

Pose Estimation

Detect and map human joints (keypoints) for activity tracking or virtual applications.

Method: estimatePose(bytes)

Core Model: SimplePose

DJL Domain: Computer Vision (Detection)

😮

Emotion Analysis

Classify the dominant facial emotion (e.g., happy, sad, neutral) in a detected face.

Method: analyzeEmotions(bytes)

Core Model: FER Model

DJL Domain: Computer Vision (Classification)

✂️

Image Segmentation

Pixel-level classification to delineate object boundaries and shapes within an image.

Method: segmentImage(bytes)

Core Model: U-Net / DeepLabV3

DJL Domain: Computer Vision (Segmentation)

🚗

License Plate Detection

Specialized detection for localizing vehicle license plates for automated systems.

Method: detectLicensePlate(bytes)

Core Model: MobileNet SSD (Tuned)

DJL Domain: Computer Vision (Detection)

Image Quality Assessment

Non-reference scoring of image aesthetics or technical quality (e.g., blur, noise).

Method: imageQuality(bytes)

Core Model: NIMA (Aesthetics)
